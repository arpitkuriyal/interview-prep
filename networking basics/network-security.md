# Network Security Interview Prep (SDE Intern / Full-Time)

---

## 1. What is Network Security?

**Answer:**
Network security is the practice of protecting a network and its data from unauthorized access, misuse, or attacks.

Simple:

> It ensures safe communication between systems over a network.

---

## 2. CIA Triad in Network Security

**Answer:**

* **Confidentiality** → Data is not exposed
* **Integrity** → Data is not modified
* **Availability** → Network is accessible

Example:

> Secure banking transactions require all three.

---

## 3. What is a Firewall?

**Answer:**
A firewall monitors and controls incoming and outgoing network traffic based on rules.

Types:

* Network Firewall
* Application Firewall

Simple:

> Acts as a security gate between trusted and untrusted networks.

---

## 4. What is VPN?

**Answer:**
A VPN (Virtual Private Network) creates an encrypted connection over the internet.

Benefits:

* Secure communication
* Hides IP address

---

## 5. What is IDS vs IPS?

### IDS (Intrusion Detection System):

* Detects suspicious activity

### IPS (Intrusion Prevention System):

* Detects and blocks attacks

**Answer:**

> IDS alerts, IPS takes action.

---

## 6. What is SSL/TLS?

**Answer:**
SSL/TLS are protocols used to encrypt communication over a network.

Used in:

* HTTPS

Ensures:

* Data privacy
* Secure communication

---

## 7. What is a Proxy Server?

**Answer:**
A proxy acts as an intermediary between client and server.

Uses:

* Security
* Anonymity
* Caching

---

## 8. What is NAT (Network Address Translation)?

**Answer:**
NAT converts private IP addresses into public IP addresses.

Why?

> Allows multiple devices to share one public IP.

---

## 9. What is Port Scanning?

**Answer:**
Port scanning is used to find open ports on a system.

Used by:

* Attackers → find vulnerabilities
* Admins → security testing

---

## 10. What is Packet Sniffing?

**Answer:**
Capturing and analyzing network packets.

Risk:

> Can expose sensitive data if unencrypted.

---

## 11. What is a DDoS Attack?

**Answer:**
Distributed Denial of Service floods a server with traffic to make it unavailable.

Impact:

* Server crash
* Downtime

---

## 12. What is Man-in-the-Middle (MITM)?

**Answer:**
An attacker intercepts communication between two parties.

Example:

> Public WiFi attack stealing data.

---

## 13. What is ARP Spoofing?

**Answer:**
Attacker sends fake ARP messages to link their MAC address with a legitimate IP.

Result:

> Traffic gets redirected to attacker.

---

## 14. What is DNS Spoofing?

**Answer:**
Attacker redirects users to fake websites by corrupting DNS responses.

---

## 15. What is Secure Socket Layer Attack (SSL Stripping)?

**Answer:**
Downgrades HTTPS connection to HTTP.

Risk:

> Data becomes unencrypted.

---

## 16. What is Network Segmentation?

**Answer:**
Dividing a network into smaller parts for security.

Benefit:

> Limits attack spread.

---

## 17. What is Access Control?

**Answer:**
Restricting who can access network resources.

Types:

* Role-based
* Mandatory
* Discretionary

---

## 18. What is Zero Trust Network?

**Answer:**
No user or device is trusted by default.

Rule:

> Always verify before granting access.

---

## 19. What is TLS Handshake?

**Answer:**
Process of establishing a secure connection.

Steps:

1. Client hello
2. Server hello
3. Certificate exchange
4. Key exchange
5. Secure connection established

---

## 20. What is Forward Secrecy?

**Answer:**
Even if encryption keys are compromised, past communication remains secure.

---

## 21. What is Honeypot?

**Answer:**
A fake system used to attract attackers and study their behavior.

---

## 22. What is SIEM?

**Answer:**
Security Information and Event Management collects and analyzes security data.

---

## 23. What is Network Hardening?

**Answer:**
Strengthening network security by reducing vulnerabilities.

Examples:

* Disable unused ports
* Strong passwords
* Regular updates

---

## 24. What is Rate Limiting?

**Answer:**
Restricts number of requests to prevent abuse and attacks.

---
